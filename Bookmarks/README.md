# Расширения для отображения закладок


## Объединение функций расширении

**Каким должно быть идеальное расширение?**

* Открытие закладки в новой вкладке должно происходить по нажатию средней или левой кнопки мыши, удерживая клавишу Ctrl
* Фильтрация закладок по тэгам (меткам)
* Поисковая панель с поиском по тэгам, ссылке или заголовку/описанию. И здесь же показ месторасположения среди остальных. (Ctrl + F — сочетание клавиш для вызова поисковой панели)
* Тэгирование закладок, присвоение им меток
* Поиск закладки по мере ввода
* Изменяемая цветовая схема (светлая и тёмная)
* Отображение папки (категории) в результатах поиска
* Отображение закладок в виде древовидного меню
* Drag&Drop для папок/категорий/директорий
* Перетаскивание/перемещение закладок из одной папки в другую
* Перетаскивание/перемещение папок с закладками
* Скрытие закладок или папки с закладками
* При переходе по закладке оставлять всплывающее меню открытым
* Закрывать неиспользуемые открытие вкладки при навигации
* Левой кнопкой мыши открывать ссылки в текущей вкладке
* Средней кнопкой мыши открывать ссылки в качестве новой активной вкладки
* Чтобы открыть закладку в новой вкладке и сразу на неё переключиться, нажмите среднюю кнопку мыши, удерживая клавишу Shift
* Чтобы открыть закладку в новом окне, нажмите левую кнопку мыши, удерживая клавишу Shift
* Чтобы открыть все закладки в папке, нажмите среднюю кнопку мыши
* Открывать ссылки 2-ым щелчком, а не одинарным, тогда ссылками было бы удобнее манипулировать (перетаскивать, выделять несколько...)

При работе с папками теперь возможно два сценария поведений - помещение закладок в папку (при перетаскивании на текст названия папки) и перемещение закладок на позицию следующую за выбранной папкой.

Кнопка отчистки строк поиска

- Внешний вид:
	- Возможность задать свой размер окна
	- Возможность задать свой безопасный шрифт
	- Возможность задать свой стиль (css)

Добавить в контекстное меню папки с закладками пункты:
- Открыть все закладки
- Открыть все закладки в новом окне
- Открыть все закладки в новом приватном окне/инкогнито(в режиме инкогнито)
- Упорядочить по заголовкам
- Удалить
- Переименовать
- Добавить/Создать папку

Добавить в контекстное меню закладки пункты:
- Открыть закладку (в этой вкладке)
- Открыть в новой вкладке
- Открыть в новом окне
- Открыть в новом приватном окне (в режиме инкогнито)
- Удалить
- Переименовать
- Редактировать ссылку (адрес/url)

Какое расширение фокрнуть и на его основное создать новое дополнив необходимым функционалом решать вам.
