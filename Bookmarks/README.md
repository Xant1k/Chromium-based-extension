# Объединение функций расширении
***

**1. Закладки**

https://addons.opera.com/extensions/details/bookmarks-2/

https://chrome.google.com/webstore/detail/mhophkifmlkobgkeahlhcdnjadcpmlbp

Доступ к закладкам, истории и приложениям.

**Внимание! Требует много прав. Одна из прошлых версий содержала вредоносный код.**

**Особенности:**
* Настройка открытия вкладок кликом по ним: клик левой или средней кнопкой мыши
* При открытии папки закрывать другие
* Импорт и экспорт закладок
* Просмотр популярных закладок двойным кликом по значку расширения
* Поиск среди закладок.
* Запоминать позицию прокрутки

В настройках расширения есть опции:
* Действие после открытия закладки в: Новой вкладке, Текущей вкладке (закрыть окно расширения или оставить открытым)
* Изменить иконку (можно выбрать произвольную)
* Ширина окна
* Размер шрифта

- Операции над папкой с закладками:
	- Добавить папку
	- Добавить страницу
	- Открыть все закладки
	- Открыть все закладки в новом окне
	- Открыть все закладки в режиме инкогнито
	- Сортировать: по алфавиту, дате
	- Переименовать папку
	- Удалить папку

Операции над закладкой: Открыть в новой вкладке Открыть в закреплённой вкладке Открыть в новом окне Открыть в режиме инкогнито Редактировать закладку Удалить закладку

**3. Bookmarkie**

https://chrome.google.com/webstore/detail/bookmarkie/ahlphbdcaacfhkiajebghpngknafklbj

https://github.com/angusjune/bookmarkie

Расширение для просмотра закладок в виде древовидной структуры. Есть поиск по имемни среди закладок и выполнение различных действий над ними:

Раскрытая папка отображается светлым цветом, свёрнутая - тёмным.

- Операции над папкой с закладками:
	- Открытие всех закладок
	- Открытие всех закладок в новом окне
	- Открытие всех закладок в приватном окне
	- Редактировать (изменение имени)
	- Удаление

- Операции над закладкой:
	- Открыть в новой вкладке
	- Открыть в новом окне
	- Открыть в приватном окне Редактировать (изменить имя) Удалить

В настройках расширения есть опции:
* Всегда открывать закладки в новых вкладках
* Щелчок средней кнопкой мыши или ctrl+клик открывает закладку в новой фоновой вкладке
* Закрывать неиспользуемые папки при открытии папки
* Всплывающее окно остается открытым при открытии закладок
* Показывать подтверждение при открытии большого количества закладок из папки
* Запомнить предыдущее состояние (положение прокрутки и открытые папки)
