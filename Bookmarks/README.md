# Расширения для отображения закладок


## Объединение функций расширении

**Каким должно быть идеальное расширение?**

* Открытие закладки в новой вкладке должно происходить по нажатию средней или левой кнопки мыши, удерживая клавишу Ctrl
* Фильтрация закладок по тэгам (меткам)
* Поисковая панель с поиском по тэгам, ссылке или заголовку/описанию. И здесь же показ месторасположения среди остальных. (Ctrl + F — сочетание клавиш для вызова поисковой панели)
* Тэгирование закладок, присвоение им меток
* Поиск закладки по мере ввода
* Изменяемая цветовая схема (светлая и тёмная)
* Отображение папки (категории) в результатах поиска
* Отображение закладок в виде древовидного меню
* Drag&Drop для папок/категорий/директорий
* Перетаскивание/перемещение закладок из одной папки в другую
* Перетаскивание/перемещение папок с закладками
* Скрытие закладок или папки с закладками
* При переходе по закладке оставлять всплывающее меню открытым
* Закрывать неиспользуемые открытие вкладки при навигации
* Левой кнопкой мыши открывать ссылки в текущей вкладке
* Средней кнопкой мыши открывать ссылки в качестве новой активной вкладки
* Чтобы открыть закладку в новой вкладке и сразу на неё переключиться, нажмите среднюю кнопку мыши, удерживая клавишу Shift
* Чтобы открыть закладку в новом окне, нажмите левую кнопку мыши, удерживая клавишу Shift
* Чтобы открыть все закладки в папке, нажмите среднюю кнопку мыши
* Открывать ссылки 2-ым щелчком, а не одинарным, тогда ссылками было бы удобнее манипулировать (перетаскивать, выделять несколько...)

Левый клик откроет закладку в новой вкладке
Клик средней кнопкой мыши открывает закладки в новой фоновой вкладке
Закрыть неиспользуемые папки когда открыта папка
Оставить окно открытым при открытий (ЛКМ) закладок
Подтверждающее сообщение при открытии большого количества закладок из папки
Запомнить предыдущую позицию (позицию прокрутки и открытие папки)

При работе с папками теперь возможно два сценария поведений - помещение закладок в папку (при перетаскивании на текст названия папки) и перемещение закладок на позицию следующую за выбранной папкой.

Кнопка отчистки строк поиска

- Внешний вид:
	- Возможность задать свой размер окна
	- Возможность задать свой безопасный шрифт
	- Возможность задать свой стиль (css)

Добавить в контекстное меню папки с закладками пункты:
- Открыть все закладки
- Открыть все закладки в новом окне
- Открыть все закладки в новом приватном окне/инкогнито(в режиме инкогнито)
- Упорядочить по заголовкам
- Удалить
- Переименовать
- Добавить/Создать папку

Добавить в контекстное меню закладки пункты:
- Открыть закладку (в этой вкладке)
- Открыть в новой вкладке
- Открыть в новом окне
- Открыть в новом приватном окне (в режиме инкогнито)
- Удалить
- Переименовать
- Редактировать ссылку (адрес/url)

Какое расширение форкнуть и на его основное создать новое дополнив необходимым функционалом решать вам.

**1. Закладки**

https://addons.opera.com/extensions/details/bookmarks-2/

https://chrome.google.com/webstore/detail/mhophkifmlkobgkeahlhcdnjadcpmlbp

**Внимание! Требует много прав. Одна из прошлых версий содержала вредоносный код.**

**Особенности:**
* Настройка открытия вкладок кликом по ним: клик левой или средней кнопкой мыши
* При открытии папки закрывать другие
* Импорт и экспорт закладок
* Просмотр популярных закладок двойным кликом по значку расширения
* Поиск среди закладок.
* Запоминать позицию прокрутки

- Операции над папкой с закладками:
	- Добавить папку
	- Добавить страницу
	- Открыть все закладки
	- Открыть все закладки в новом окне
	- Открыть все закладки в режиме инкогнито
	- Сортировать: по алфавиту, дате
	- Переименовать папку
	- Удалить папку

- Операции над закладкой:
	- Открыть в новой вкладке
	- Открыть в закреплённой вкладке
	- Открыть в новом окне
	- Открыть в режиме инкогнито
	- Редактировать закладку
	- Удалить закладку

В настройках доступны следующие опции:
* Действие после открытия закладки в: Новой вкладке, Текущей вкладке (закрыть окно расширения или оставить открытым)
* Изменить иконку (можно выбрать произвольную)
* Ширина окна
* Размер шрифта

При 20к закладках время между кликом на иконку расширения и появлением меню занимает ~2 сек. Скорость открытия папки с 2.5к закладками составляет ~ 2-3 сек, прокрутка медленная; с 15к ~ 10 сек, при этом прокрутка заметно лагает.

	- Открыть в новой вкладке
	- Открыть в новом окне

В настройках доступны следующие опции:
* Всегда открывать закладки в новых вкладках

* Показывать подтверждение при открытии большого количества закладок из папки
* Запомнить предыдущее состояние (положение прокрутки и открытые папки)

При 20к закладках время между кликом на иконку расширения и появлением меню занимает не более одной секунды. Скорость открытия папки с 2.5к закладками составляет ~ 2-3 сек, прокрутка плавная; с 15к ~10 сек, при этом прокрутка лагает.

Время открытия меню при клике на иконку не занимает больше одной секунды. Скорость открытия папки с 15к закладок ~10 сек. Прокрутка внутри папок с большим закладок лагает. Скорость открытия папки с 2.5к закладок составляет 2-3 сек, при этом прокрутка внутри плавная.

	- Открыть в фоновой вкладке
	- Открыть в новом окне
	- Открыть в режиме инкогнито
	- Изменить... (переименовать название и адрес)

В настройках доступны следующие опции:

- General:
	- Директория расширений по умолчанию (панель закладок или другие закладки или корзина, в общем отображать по умолчанию нужную папку)
	- Скрыть корневую директрию (скрывать папки выборочно)
	- Разрешить поиск в (заголовку и адресу, только заголовку)
	- Максимальное количество закладок в результатах поиска
	- Показывать подсказки с полной информацией о закладке
	- Предупреждать при открытии нескольких закладок
	- Запоминать последнюю позицию

- UI:
	- Ширина панели
	- Размер шрифта
	- Шрифт

- Control:
	- Левый клик для открытия закладки в:
	- Ctrl / CMD + Левый клик для открытия закладки в
	- Shift + Левый клик для открытия закладки в
	- Средний клик для открытия закладки в
	- Открывать папки левым кликом: да/нет (этот пункт нужен чтобы автоматически папки не раскрывались)
	
Первые 4 опции содержут следующие изменения поведения:
-Новая вкладка
-Текущая вкладка
-Текущая вкладка (без закрытия)
-Фоновая вкладка
-Фоновая вкладка (без закрытия)
-Новое окно
-Окно инкогнито

При 20к закладках между кликом на иконку расширения и появлением меню проходит ~1 сек. Прокрутка внутри папок с большим закладок быстрая и плавная.
